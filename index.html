Prework Study Guide

✨ Open the Console (ctrl + shift + i) to see what's happening ✨

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The meta element contains information useful for the search engines
  • The footer element often contains copyright information

CSS

  • CSS is used to control the website's style (looks)
  • CSS stands for Cascading Style Sheets
  • CSS style sheet needs to be linked in the main HTML file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is used for distributed version control
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A : stages changes
  • git commit -m "my comment" : commits to repo
  • git pull origin main : updates current branch
  • git push origin name1/name2-name3 : pushes all changes

JavaScript

  • JavaScript helps make websites interactive
  • A variable is a named container that allows us to store data in our code. It looks like this: var variableName = [array]
  • Control flow is the order in which a computer executes code in a script
  • Functions consist of "function" keyword, function name, (), {function body}
  • Functions need to be called to run
